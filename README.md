# Jakkur Lake Water Mapping 🌊

Monitoring and mapping of Jakkur Lake, Bangalore using 
satellite imagery and GIS techniques.

## Project Overview
Analysis of Jakkur Lake's water body extent and health 
using Sentinel-2 satellite data and NDWI 
(Normalized Difference Water Index).

## Technologies Used
- QGIS
- Sentinel-2 Satellite Imagery
- NDWI (Normalized Difference Water Index)
- Python (GeoPandas, Rasterio)
- GeoPackage (.gpkg) format

## Methodology
1. Acquired Sentinel-2 satellite imagery
2. Calculated NDWI to identify water bodies
3. Mapped lake boundary and extent
4. Analyzed water body changes

## Files
- `Jakkur_Lake_NDWI.qgz` → QGIS project with NDWI analysis
- `Jakkur_Lake_NDWI_RECOVERY.qgz` → Recovery analysis project
- `scripts/` → Python processing scripts
- `report/` → Project report and findings
- `outputs/` → Generated maps and outputs
